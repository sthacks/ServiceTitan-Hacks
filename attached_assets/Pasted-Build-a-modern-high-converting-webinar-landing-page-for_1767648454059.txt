Build a modern, high-converting webinar landing page for a webinar titled "The Invisible Labor Market".

Goal
- Drive webinar registrations.
- Present the problem and promise clearly.
- Make the registration CTA obvious and repeated.

Tech
- Use Next.js (App Router) + TypeScript + Tailwind CSS.
- Single public route: "/" with one page component.
- No backend required.
- Use semantic HTML, accessible labels, and good Lighthouse scores.

Brand + Style
- Clean, modern SaaS look.
- White background, subtle light gray sections, dark text.
- Accent color: #6D28D9 (purple). Secondary accent: #111827 (near-black).
- Rounded cards, soft shadows, generous spacing.
- Typography: system font stack (or Inter if available).
- Responsive: mobile-first, looks great at 390px, 768px, 1024px, 1440px.
- Sticky top bar with CTA on desktop, collapses cleanly on mobile.

Hero CTAs:
- Primary: "Register free"

Registration form behavior (important)
- The page should support two registration modes:
  Mode A (default): simple on-page form that redirects to a Zoom registration link with query params.
  Mode B (optional): embed an iframe for Zoom registration if I paste an embed URL.
Implementation details:
- Create a config object at the top of the page:
  - webinarTitle
  - webinarDate (string)
  - webinarTime (string)
  - timezone (string)
  - zoomRegistrationUrl (string placeholder)
  - zoomEmbedUrl (string optional, can be empty)
- If zoomEmbedUrl is non-empty, show an iframe embed in the registration card.
- If zoomEmbedUrl is empty, show a local form with:
  - First name
  - Last name
  - Email
On submit:
- Validate required fields.
- Build a redirect URL to zoomRegistrationUrl, appending:
  - first_name, last_name, email as query params (use encodeURIComponent).
- Then window.location.href to that URL.
Also:
- Add a checkbox: "Send me the templates and reminders" (checked by default). If checked, include a query param "opt_in=1" else "opt_in=0".

Other requirements
- Add smooth scrolling for anchor links.
- Add section IDs for navigation: learn, agenda, speakers, faq.
- Add basic SEO metadata: title, description, Open Graph placeholders.
- Add a small footer with links: "Privacy", "Terms", and "Contact" (placeholders).
- Include a "Back to top" button that appears after scrolling 600px.

Deliverables

- Keep the code clean and editable.
