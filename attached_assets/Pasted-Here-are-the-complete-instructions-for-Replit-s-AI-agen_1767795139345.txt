## Lead Webhook Integration Instructions for Replit

### Available Company Slugs

| Company | Slug |
|---------|------|
| Contractor Commerce | `contractor-commerce` |
| ServiceTitan Hacks | `servicetitan-hacks` |
| SmartAC | `smartac` |
| Trade Runner | `traderunner` |
| Volca.AI | *(no slug set)* |
| Wink Toolbox | `wink-toolbox` |

---

### Webhook Endpoint

```
POST https://imnhusloafhxccznjelj.supabase.co/functions/v1/create-lead
```

### Required Headers

```json
{
  "Content-Type": "application/json",
  "x-api-key": "[YOUR_LEADS_WEBHOOK_API_KEY]"
}
```

### Request Body (JSON)

| Field | Required | Description |
|-------|----------|-------------|
| `company_slug` | Yes* | Company slug from the table above (e.g., `"smartac"`) |
| `company_id` | Yes* | Alternative to slug - use the UUID directly |
| `name` | Yes | Lead's full name |
| `email` | No | Lead's email address |
| `phone` | No | Lead's phone number |
| `company_name` | No | Lead's company/business name |
| `source` | No | Where the lead came from (e.g., `"Website Form"`, `"Landing Page"`) |
| `source_details` | No | Additional source info (e.g., page URL, campaign name) |
| `notes` | No | Any additional notes or form message content |

*Either `company_slug` OR `company_id` is required (not both)

### Example Request Body

```json
{
  "company_slug": "smartac",
  "name": "John Smith",
  "email": "john@example.com",
  "phone": "555-123-4567",
  "company_name": "Smith HVAC Services",
  "source": "Website Form",
  "source_details": "Contact page - smartac.com/contact",
  "notes": "Interested in learning more about the platform"
}
```

### Example JavaScript/Node.js Code for Replit

```javascript
async function submitLeadToWebhook(formData) {
  const response = await fetch(
    'https://imnhusloafhxccznjelj.supabase.co/functions/v1/create-lead',
    {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'x-api-key': process.env.LEADS_WEBHOOK_API_KEY  // Store in Replit Secrets
      },
      body: JSON.stringify({
        company_slug: 'smartac',  // Change to your company slug
        name: formData.name,
        email: formData.email,
        phone: formData.phone,
        company_name: formData.companyName,
        source: 'Website Form',
        source_details: window.location.href,
        notes: formData.message
      })
    }
  );

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.error || 'Failed to submit lead');
  }

  return await response.json();
}
```

### Response Codes

| Status | Meaning |
|--------|---------|
| `201` | Lead created successfully |
| `400` | Missing required field (name, company_slug/company_id) |
| `401` | Invalid or missing API key |
| `404` | Company not found (invalid slug) |
| `500` | Server error |

### Replit Setup Steps

1. Store the API key in Replit's **Secrets** tab as `LEADS_WEBHOOK_API_KEY`
2. Map your form fields to the webhook body fields
3. Call the webhook on form submission
4. Handle success/error responses appropriately
