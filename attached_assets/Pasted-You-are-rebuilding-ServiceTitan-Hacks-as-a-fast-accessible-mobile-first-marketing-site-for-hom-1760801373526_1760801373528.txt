You are rebuilding **ServiceTitan Hacks** as a fast, accessible, mobile-first marketing site for home-service contractors. Produce production-ready HTML, CSS, and minimal JS with clean, semantic structure. No frameworks. No external CDNs except Google Fonts. Ship a complete, multi-page site with navigation, footer, and shared styles.

## Brand
- Name: ServiceTitan Hacks
- Voice: direct, contractor-focused, zero fluff
- Colors:
  - Primary Red: #ED254E
  - Deep Red: #C1124F
  - Dark Gray: #1F1F1F
  - Mid Gray: #6B7280
  - Black: #000000
  - White: #FFFFFF
- Typography: "Oxygen" for headings, "Inter" for body (load from Google Fonts). Tight, high-contrast type scale.

## Global requirements
- Fully responsive, mobile-first, CLS-safe layout
- WCAG 2.1 AA contrast
- Prefetch fonts, use font-display:swap
- Meta + OpenGraph + Twitter cards per page
- JSON-LD schema where relevant (Organization, Website, Article, PodcastEpisode, Product)
- Image `<img>` with `width/height` and `loading="lazy"`
- Accessible nav with keyboard focus states
- Reusable components in a `/assets/` folder: buttons, cards, badges, grid, section wrappers
- Simple, dependency-free JS for mobile menu, accordions, tabbed content, and form validation
